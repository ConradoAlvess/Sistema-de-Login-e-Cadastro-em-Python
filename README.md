# Sistema de Login e Cadastro em Python

Projeto desenvolvido para simular um sistema de autenticação com cadastro e login de usuários.

## Funcionalidades
- Cadastro de usuários
- Login com validação
- Armazenamento em arquivo JSON

## Tecnologias
- Python
- JSON

## Como executar
1. Instale o Python
2. Execute:
python main.py

## Objetivo
Praticar conceitos de autenticação, estrutura de dados e persistência de dados.
